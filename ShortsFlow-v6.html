🔗 Connect Your Google Sheet
Follow these steps once to connect your sheet. After that, just fill the sheet and editors see it automatically.
1
Create your Google Sheet
Go to sheets.google.com → New blank sheet. Make sure the tab at the bottom is named Sheet1. Your data will start from row 11 (rows 1-10 are for headers and notes).
2
Add column headers in row 10
Click cell A10 and type these across the row:
A10: SHORT NO.  |  B10: Team Name  |  C10: HOOK/SEO  |  D10: CREATIVE  |  E10: READY?  |  F10: VPS COMMENT  |  G10: (blank)  |  H10: Internal
Then select E11:E200 → Insert → Checkbox (makes column E proper tick boxes)
3
Install the Apps Script
In your Sheet → Extensions → Apps Script → delete all existing code → paste the ShortsFlow-AppsScript.js file → Save (Ctrl+S).

Then select setupExtraColumns from the dropdown → click ▶ Run → authorize when asked.
4
Deploy as Web App
In Apps Script → Deploy → New deployment → Web App
Set: Execute as: Me  |  Who has access: Anyone
Click Deploy → copy the URL
5
Paste your Web App URL here & connect
Paste the URL below. Click Test first to make sure it works, then Connect.
